Driving education module

I created this game prototype as part of thesis project for my M.Sc. at Universiteit Utrecht in the Netherlands.The goal was to test how competition and perspective influence learning in the context of traffic education.

For this I created a game where one is presented with a situation between two cars at an intersection. The player has to decide what the correct action is between Go and Yield, according to road regulations.

This project was made using Unity 5. The whole prototype was programmed by me, from analysis to code deployment. All art assets were obtained from the web

The main things I learned from the creation of this experiment tool were:

  • Building and assembling big scenes
  • Deploying a game for WebGL
  • Implementation of an AI adversary modeled from human interactions
  • Recolletion of gameplay data
  • Interaction with databases through web services
  • Cloud Content Delivery through AssetBundles
